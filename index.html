關於我

我目前主要專注在前端開發、教學與接案合作，日常工作內容包含切版、RWD 介面實作、元件化結構設計，以及協助把設計稿穩定落地到瀏覽器中。平常會大量使用 HTML 與 CSS，也會依專案需要搭配現代前端工具與工作流程。

在合作與教學上，我特別重視「把事情講清楚」：習慣先釐清需求、拆解問題，再用對方聽得懂的方式說明技術細節。比起炫技，我更在意的是溝通順暢、可維護性與長期穩定，讓設計、產品與工程彼此都有安全感。

  • 可以協助檢視既有版面結構，提出 HTML / CSS 的重構與優化建議。
  • 擅長把複雜的需求拆成明確的開發步驟，做成好溝通、好追蹤的任務清單。
  • 喜歡教學與陪跑，適合協助初學者或轉職者建立穩固的前端基礎概念。
